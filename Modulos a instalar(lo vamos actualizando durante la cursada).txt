NodeJS (instalo una sola vez global a la pc):

Modulos que vamos instalando:
npm i axios (permite hacer solicitudes http)
npm i sha1 (encriptacion de passwords)
npm i @hapi/joi (validaciones)
npm i mysql (conexion con mysql)
npm i knex (querys simplificadas de sql SQL query builder)
npm i dotenv (variables de entorno)
npm i jsonwebtoken (JWT tokens)
